Step1.远程桌面连接云服务器

按要求在网上购买阿里云，腾讯云，百度云的服务器，启用并记住公网IP，桌面->开始菜单->windows附件->远程桌面链接->输入你的IP地址和实例密码，进行连接。
连接过程中，你可能会遇到以下情况：提示“您的凭据不工作 之前用于连接到某凭据无法工作。请输入新凭据。”
首先先确认自己的IP地址和密码是否正确。
若无误，请参考以下解决办法：https://blog.csdn.net/huyuyang6688/article/details/49077665

找不到gpedit.msc组策略参考解决办法：
https://baijiahao.baidu.com/s?id=1608865325022611543&wfr=spider&for=pc

Step3.添加ftp服务

这里给大家分享通过FileZilla在阿里云主机上搭建ftp服务器

Step4.搭建mysql数据库

①安装mysql数据库
MySQL安装教程：https://blog.csdn.net/bobo553443/article/details/81383194
在Mysql安装过程中，可能出现”no compatible servers were found”错误，这是因为缺少c++运行库造成的，我们需要去安装所需要的运行库，在安装界面它会告诉你缺少什么，一般是Visual C++ 2013 and Visual C Redistributable Package和Visual C++ 2015 and Visual C Redistributable Package（最新版本Mysql需要）
2013:https://www.microsoft.com/zh-cn/download/details.aspx?id=40784(这个貌似要装32位的)
2015:https://www.microsoft.com/en-us/download/details.aspx?id=48145
②用本地Navicat进行连接
注意：千万不要忘记在云服务器控制台开放3306端口。开放方法与上面类似。
连接出现的错误解决办法：
（1）error1130:https://blog.csdn.net/luoxiping1/article/details/80010976
（2）error1251：https://blog.csdn.net/qq_40605167/article/details/81253976
